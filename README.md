# fontend_homework_fis3Baidu
利用Gulp或者FIS重组百度首页

利用Gulp或者FIS重新组织你的百度首页样式和JS，然后达到工程化可编译状态。
1.安装Gulp或者FIS
2.按照框架要求编译百度首页达到性能最优状态。
3.通过本作业了解一种构建工具使用的基本原理即可（不同企业构建工具可能不一样，但原理基本一样），并学会独立查文档去学习新知识。
（视频中教学用的是FIS2，如果有同学用的是FIS3也可（两个不同版本）。
